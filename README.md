# Booqable helper platform

## Local development

### Prerequisites

- **Node.js**: Next.js 16 requires Node **>= 20.9.0**.
- **pnpm**: This repo uses `pnpm-lock.yaml`.

### Install

```bash
pnpm install
```

### Run

```bash
pnpm dev
```

Then open `http://localhost:3000`.

### Lint

```bash
pnpm lint
```

### Build

```bash
pnpm build
pnpm start
```
